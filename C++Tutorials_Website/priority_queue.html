Using the sort algorithm

In this header file, we define our own container-type, a custom "PriorityQueue" class, built on top of an underlying vector. We began with the template declaration in order to allow for the underlying vector to store generic data types. We implement a private helper method called sortItems() which sorts the elements of the vector from least to greatest. The public method push() calls this sortItems() method each time after a new item is pushed onto the end of the vector in order that the elements in the vector maintain their order from least to greatest.



Other methods implemented include: size(), which returns the number of elements in the vector, top(), which returns the item at the front of the queue without removing it (in this case, this item also happens to be the largest item), and pop(), which removes the item at the front of the queue(i.e. the largest item in the vector)

#ifndef _PRIORITY_QUEUE_H
#define _PRIORITY_QUEUE_H

#include <algorithm>
#include <vector>
#include "PriorityQueueInterface.h"

using namespace std;

template <class ItemType>
class PriorityQueue(){
private:
vector <ItemyType> items;
void sortItems() {
sort(items.begin(),items.end());
}
public:
PriorityQueue(){};
void push( const ItemType& newItem){
items.push_back(newItem);
sortItems();
}

void pop(){
auto iter_end = items.end(); iter_end--;
items.erase(iter_end);
}

unsigned int size() const{
return items.size();
}

bool empty() const{
return items.empty();
}

ItemType top() const{
auto iter_end = items.end(); --iter_end;
return *iter.end;
}

#endif
